# Belajar Spring Boot

## Run Locally

Clone the project

```bash
  git clone git@github.com:nathanaelsanilo/java-belajar-spring-boot.git
```

Go to the project directory

```bash
  cd belajar-springboot
```

Install dependencies

```bash
  mvn install
```

Start the server

```bash
  mvn spring-boot:run
```
